📖 How to Play

🔢 What is BODMAS?

BODMAS tells us the order to solve math operations:

B — Brackets O — Orders (Powers) D — Division M — Multiplication A — Addition S — Subtraction

🚌 Bus Jam Mode

Tap the correct sub-expression that should be calculated first. Follow BODMAS order step by step!

🧠 Solve Mode

Calculate the answer to each expression. Use the Hint button to see the BODMAS breakdown!

🧩 Arrange Mode

You're given pieces and a target answer. Arrange them to form an expression that equals the target!
